# Application

The bambulabstatuschecker is a small webserver that allows subscribing to locallly networked bambulab printer's MQTT datastreams, and displaying them on a web browser. The web page is minimalistic and is capable of showing more than one printer's status in tabular format. The UI supports entering and editing connection information for added printers. The webserver is intended to be ran locally on a users windows machine.

# GIT

The bambulabstatuschecker should be managed using GIT, main is located in GITHUB. 

# MQTT

The MQTT server should be generic enough to support future MQTT device monitoring with minor updates to the UI
